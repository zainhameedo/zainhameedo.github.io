Economic Data Forecasting

My implementation of cutting-edge forecasting models on key St. Louis Fed economic data

Features

  • Real-time economic data from FRED API
  • Interactive parameter adjustment for models including ARIMA and XGBoost
  • Multiple target variables (Energy Consumption, Retail Sales, GDP)
  • Visual comparison of different forecasting methods
  • Confidence intervals for predictions

Technical Stack

  • Python with Dash for the web interface
  • Pandas and NumPy for data manipulation
  • Statsmodels for ARIMA modeling
  • XGBoost for machine learning predictions
  • FRED API for economic data
  • Plotly for interactive visualizations
